# Contacts
Данная программа - консольная записная книжка. Для каждого контакта хранит фамилию, имя и номер телефона.
При запуске программы без аргументов выводится список контактов.

```
$ contacts
79098956182	Иванов	Сергей
78561289511	Сидоров	Владимир
79116551887	Ким	Вася
```

При запуске программы с флагом `-tel num` отображается контакт с номером телефона num (ожидается 11 символов).

```
$ contacts -tel 79098956182
79098956182	Иванов	Сергей
```

При запуске программы с флагом и аргументом `-add "num name lastname"` создается новый контакт.

```
$ contacts -add "78951236699 Васильков Никита"
New contact created
```

При запуске программы с флагом `-help` выводится подсказка по использованию программы.
Контакты хранятся в текстовых файлах в директории data.
